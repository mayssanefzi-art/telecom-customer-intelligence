# Telecom Customer Intelligence Platform

End-to-end data science project combining customer segmentation,
churn prediction, and fraud detection on real telecom data.

## Modules
- **Module 1 — Customer Segmentation**: KMeans clustering on IBM Telco data
- **Module 2 — Churn Prediction**: XGBoost + SHAP explainability (AUC: 0.XX)
- **Module 3 — Fraud Detection**: SMOTE + XGBoost on imbalanced data (AUC: 0.XX)

## Tech Stack
Python · Pandas · Scikit-learn · XGBoost · SHAP · Streamlit · Plotly
